# phpSocial Statistics Plugin
The Statistics Plugin for [phpSocial â€“ Social Network Platform](http://bit.ly/2Na83bV) allows you to track how many users view your website, their browser and operating system. It also include monthly statistics and daily visits variation analysis. If you want advanced information about a specific day, the plugin offer advanced information for your desired date, including charts, information tables and visits tracking.

# 1. Installation
1. Clone the repository
2. Copy to your website `plugins` folder the folder `plugins` from the clone.
3. Activate on your admin panel.
